Bootstrap

Bootstrap is a collection of free and open-source tools for building responsive website applications. It is used HTML, CSS, and JavaScript framework for creating mobile-first, websites. It resolves many issues that we previously encountered, one of which was..... browser compatibility issue. Nowadays, websites are optimised for all browsers (IE, Firefox, and Chrome) and screen sizes (Desktop, Tablets, Phablets, and Phones). All credit goes to the Bootstrap developers.
